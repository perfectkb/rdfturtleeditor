# Gettig Started

Editor is a work in progress however the code can be downloaded and started to test with below procedure;

    $ git clone <url-of-this-repo>
    $ cd <reponame>
    $ npm start

# Starting as PM2 Process

    $ cd <reponame>
    $ pm2 start "npm" --name "rdfeditor" -- start
